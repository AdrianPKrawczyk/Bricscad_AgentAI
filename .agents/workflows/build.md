# Szybka Kompilacja
1. Wyświetl komunikat: "Uruchamiam kompilację projektu V2..."
2. Użyj skilla `project-compiler`.
3. Jeśli wystąpią błędy, spróbuj je naprawić autonomicznie (max 3 próby).
4. Jeśli kompilacja zakończy się sukcesem, wyświetl: "Build Succeeded 🚀".
5. Jeśli błędy wymagają decyzji architekta, wyświetl listę błędów i poproś o instrukcje.
